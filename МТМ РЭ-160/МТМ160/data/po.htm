Программное обеспечение

    Приборы МТМ-РЭ-160, БПД-4 (RS-232), МТМ292, МТМ310И, МТМ900, БПС-2, ПЦ-12Р, модули серии МТМ4000 доступны для подключения к персональному компьютеру либо программируемому логическому контроллеру с использованием интерфейса RS-485 по одной из следующих схем.
    Максимальное количество приборов в одном сегменте сети RS-485 - 32. Максимальное расстояние между персональным компьютером и приборами 1200 м.

    Подключение приборов к персональному компьютеру


    Подключение приборов к персональному компьютеру с использованием телефонного соединения (кроме МТМ-РЭ-160 и ПЦ-12Р)

    В приборах НПП "Микротерм" используется 4 протокола обмена по интерфейсу RS-485:
    Протокол обменаРеализация в приборах
    MODBUS-RTU МТМ292, МТМ310И, МТМ900, БПС-2
    ADAM4000модули серии МТМ4000
    Протокол "Микротерм-01"МТМ-РЭ-160, БПД-4
    Протокол "Микротерм-02"МТМ292, ПЦ-12Р

    Планируется реализация протокола обмена MODBUS-RTU в приборах МТМ-РЭ-160, БПД-4, МТМ201, ПЦ-12Р и модулях серии МТМ4000.
    Протоколы "Микротерм-01", "Микротерм-02" далее развиваться не будут, но совместимость с ними вновь выпускаемых приборов будет обеспечена.
